    # 伪类

    🍜🍜🍜 当我们在使用CSS时，有时需要为元素的不同状态设置不同的样式，这时候，我们就可以使用CSS的伪类（pseudo-class）来选择不同状态的元素，也就是说，伪类是CSS中的一种选择器，它的特点是可以选择元素的某个特定状态。

    # :hover

    伪类的语法是在元素选择器后添加一个冒号（:），然后是伪类名称，比如：

    a:hover {
  color: red;
}
    1
    2
    3

    在这个例子中，我们使用了 :hover 伪类选择器，当鼠标悬停在链接上时，链接的颜色会变成红色。

    # :active

    :active 是 CSS 中的一个伪类，表示用户正在激活（点击）某个元素。

    例如，我们可以给按钮添加一个 :active 样式，以在用户点击按钮时改变按钮的样式：

    button:active {
  background-color: #00bfff;
  color: #fff;
}
    1
    2
    3
    4

    在这个例子中，当用户点击按钮时，按钮的背景颜色会变成蓝色，文字颜色会变成白色。

    :active 伪类通常用于交互性强的页面元素上，例如按钮、链接等，以提高用户体验。

    # :focus

    :focus 是 CSS 中的一个伪类，表示用户正在聚焦（选中）某个元素。

    例如，我们可以给输入框添加一个 :focus 样式，以在用户选中输入框时改变输入框的样式：

    input:focus {
  border-color: #00bfff;
  box-shadow: 0 0 10px #00bfff;
}
    1
    2
    3
    4

    ：在这个例子中，当用户选中输入框时，输入框的边框颜色会变成蓝色，同时会添加一个蓝色的阴影效果。

    # 总结

    # 结构性伪类

    结构性伪类 功能
    :root 选择文档的根元素。
    :nth-child() 选择该元素的父元素下的某个特定子元素，可接受一个参数来选择特定的子元素。
    :nth-last-child() 选择该元素的父元素下的某个特定子元素，但是从后往前数。
    :nth-of-type() 选择该元素父元素下特定类型的子元素，可接受一个参数来选择特定的子元素。
    :nth-last-of-type() 选择该元素的父元素下的某个特定类型的子元素，但是从后往前数。
    :first-child 选择该元素的父元素下的第一个子元素。
    :last-child 选择该元素的父元素下的最后一个子元素。
    :first-of-type 选择该元素的父元素下的特定类型的第一个子元素。
    :last-of-type 选择该元素的父元素下的特定类型的最后一个子元素。
    :only-child 选择该元素的父元素下仅有的一个子元素。
    :only-of-type 选择该元素的父元素下唯一的特定类型的子元素。
    :empty 选择不包含任何子元素的元素。

    # 用户动作伪类

    用户动作伪类 功能
    :link 选择未被访问过的链接。
    :visited 选择已被访问过的链接。
    :hover 选择鼠标悬停的元素。
    :active 选择鼠标按下时被点击的元素。
    :focus 选择获得焦点的元素。
    :focus-within 选择元素本身或者元素内某个元素获得焦点时的状态。

    # UI 状态伪类

    UI 状态伪类 功能
    :checked 选择被选中的表单元素。
    :disabled 选择被禁用的表单元素。
    :enabled 选择可用的表单元素。
    :indeterminate 选择未确定状态的表单元素。
    :default 选择默认被选中的表单元素。
    :valid 选择满足验证要求的表单元素。
    :invalid 选择不满足验证要求的表单元素。

    # 其他伪类

    其他伪类 功能
    :not() 选择不匹配指定选择器的元素。
    :target 选择当前活动的目标元素。
    :lang() 选择指定语言的元素。
    :focus-visible 选择通过键盘等非鼠标方式获得焦点的元素。

    需要注意的是，伪类以单冒号 : 开头，而不是双冒号 ::。

    # 伪元素

    🍛🍛🍛 除了伪类，还有一种叫做伪元素（pseudo-element）的选择器，它们可以选择元素的某个部分，并为其设置样式。

    伪类和伪元素可以在同一个选择器中使用，例如：

    a:hover::before {
  content: "→ ";
}
    1
    2
    3

    在这个例子中，当鼠标悬停在链接上时，链接前面会出现一个箭头符号。

    # ::before 和 ::after

    "::before" 和 "::after" 可以在元素的内容前面或后面插入新内容,并且可以通过CSS样式来控制它们的样式和位置。

    这两个伪元素的使用方法类似，例如：

    .example::before {
  content: "";
  display: inline-block;
  width: 20px;
  height: 20px;
  background-image: url(icon.png);
}
    1
    2
    3
    4
    5
    6
    7

    在这个例子中，我们使用了 ::before 伪元素，在 .example 元素的前面插入一个空的伪元素，并设置其为内联块级元素（display: inline-block;），并且给伪元素设置了宽度、高度和背景图像，使得它成为了一个20x20的图标。

    # ::first-line 和 ::first-letter

    "first-line" 伪元素用于向文本的首行设置特殊样式。

    "first-letter" 伪元素用于向文本的首字母设置特殊样式。

    p::first-line {
  font-size: 1.2em;
  font-weight: bold;
  color: blue;
}
    1
    2
    3
    4
    5

    在这个例子中，我们选择了一个 p 元素的第一行文本，并设置了该行文本的字体大小、字体加粗和颜色。

    注意： "first-line" 和 "first-letter" 伪元素都只能用于块级元素。

    # 总结

    伪元素 功能
    ::before 在元素的内容之前插入一个虚拟元素。
    ::after 在元素的内容之后插入一个虚拟元素。
    ::first-letter 选择元素的第一个字母，并可以通过 CSS 样式来控制该字符的样式。
    ::first-line 选择元素的第一行文本，并可以通过 CSS 样式来控制该行文本的样式。
    ::selection 选择文档中被用户选中的部分。
    ::backdrop 定义当元素处于全屏模式下时应用的样式。
    ::marker 定义列表中项目符号的样式。
    ::placeholder 定义输入框中的占位符文本的样式。
    ::spelling-error 定义输入框中的拼写错误文本的样式。
    ::grammar-error 定义输入框中的语法错误文本的样式。
    ::cue 定义文本轨道的内容，并可以通过 CSS 样式来控制该内容的样式。（通常用于视频或音频的文本轨道上）
    ::slotted 匹配被插入到一个具有 slot（插槽）的元素中的子元素，并可以通过 CSS 样式来控制这些子元素的样式。
    ::part 用于给 Shadow DOM 的组件部分分配样式，使得可以定义一个自定义元素中部分内容的样式。

    ::part 目前还不是 CSS 标准的一部分，仍处于实验阶段，而且只有部分浏览器支持。
